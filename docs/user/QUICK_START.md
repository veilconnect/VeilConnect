# VeilConnect 快速入门指南

## 🚀 5分钟快速上手

### 第一步：安装软件
```
Windows: 下载 VeilConnect-Setup.exe → 双击安装
macOS:   下载 VeilConnect.dmg → 拖拽到应用程序
Linux:   下载 VeilConnect.AppImage → 添加执行权限
```

### 第二步：生成身份
```
1. 启动 VeilConnect
2. 点击 "生成身份" 按钮
3. 等待身份生成完成
4. 点击 "开始聊天"
```

### 第三步：建立连接

#### 🔗 方式A：创建连接（主机）
```
1. 点击 "创建连接"
2. 复制生成的邀请码
3. 发送邀请码给朋友
4. 等待朋友发送响应码
5. 粘贴响应码 → 点击 "设置响应码"
6. 连接成功！开始聊天 🎉
```

#### 🔗 方式B：加入连接（客户端）
```
1. 点击 "加入连接"
2. 粘贴朋友发送的邀请码
3. 复制生成的响应码
4. 发送响应码给朋友
5. 等待连接建立
6. 连接成功！开始聊天 🎉
```

### 第四步：开始聊天
```
✅ 连接状态显示 "已连接"
✅ 在输入框输入消息
✅ 按 Enter 键或点击发送
✅ 享受端到端加密的安全聊天！
```

## 💡 使用技巧

### 🔥 热门功能
- **文件传输**：点击 📎 按钮发送文件
- **表情符号**：直接输入 😊 等表情
- **快捷键**：Enter发送，Shift+Enter换行
- **复制代码**：点击代码区域自动选中

### ⚡ 连接优化
- 使用稳定的网络环境
- 关闭VPN和代理软件
- 确保防火墙允许WebRTC
- 重新生成代码如果连接失败

### 🛡️ 安全提醒
- 不要分享私钥给任何人
- 验证对方身份后再聊天
- 定期清除聊天记录
- 使用官方下载渠道

## 🆘 遇到问题？

### 常见问题快速解决

#### ❌ 连接失败
```
1. 检查网络连接
2. 重新生成邀请码
3. 尝试使用移动热点
4. 确认代码复制完整
```

#### ❌ 无法发送消息
```
1. 确认连接状态为 "已连接"
2. 检查网络稳定性
3. 重新建立连接
4. 重启软件
```

#### ❌ 软件无法启动
```
1. 以管理员身份运行
2. 检查系统兼容性
3. 重新下载安装
4. 联系技术支持
```
